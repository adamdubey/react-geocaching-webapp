# react-geocaching-webapp

---

## Quick Start

_For Client App:_

```sh
# cd into client/ dir
cd client/

# Install dependencies
npm install

# Start the app
npm run start

# in browser, visit http://localhost:3000
# to STOP, Ctrl+C in Terminal
```

_For Server App:_

```sh
# Install dependencies
npm install

# Start the app
npm run dev

# in browser, visit http://localhost:4000
# to STOP, Ctrl+C in Terminal
```
